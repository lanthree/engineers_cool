# 58. 最后一个单词的长度

[原题链接](https://leetcode-cn.com/problems/length-of-last-word/)

总是喜欢用一个大循环解决问题:

```cpp
class Solution {
public:
    int lengthOfLastWord(string s) {
        int ans = 0;
        
        bool start_word = false;
        for (int i = s.length()-1; i >= 0; --i) {
            if (start_word) {
                if (s[i] != ' ') {
                    ans++;
                } else {
                    break;
                }
            } else {
                if (s[i] == ' ')
                    continue;
                else {
                    start_word = true;
                    ans++;
                }
            }
        }
        return ans;
    }
};
```

看了别人的代码才发现有更好的编码实现:

```cpp
class Solution {
public:
    int lengthOfLastWord(string s) {
        int ans = 0;
        
        int end = s.length()-1;
        while (end >= 0 && s[end] == ' ')
            --end;
        
        while (end >= 0 && s[end--] != ' ')
            ans++;
        
        return ans;
    }
};
```
