# 55. 跳跃游戏

[原题链接](https://leetcode-cn.com/problems/jump-game/)

简单的贪心即可，前面有一道类似的题（45. 跳跃游戏 II）：

```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int max_pos = 0;
        for (int i = 0; i <= max_pos; i++) {
            max_pos = max(max_pos, i+nums[i]);
            if (max_pos >= nums.size()-1) {
                return true;
            }
        }
        return false;
    }
};
```
