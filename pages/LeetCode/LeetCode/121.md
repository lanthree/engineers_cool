# 121. 买卖股票的最佳时机

[原题链接](https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock/)

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int ret = 0;
        int min_price = INT_MAX;
        for (int price : prices) {
            if (price < min_price) {
                min_price = price;
                continue;
            }
            ret = max(ret, price-min_price);
        }
        return ret;
    }
};
```
