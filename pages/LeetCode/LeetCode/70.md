# 70. 爬楼梯

[原题链接](https://leetcode-cn.com/problems/climbing-stairs/)

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if (n == 1) return 1;
        if (n == 2) return 2;

        int p1 = 1, p2 = 2;
        int p3 = p1 + p2;
        while(n--> 3) {
            p1 = p2;
            p2 = p3;
            p3 = p1 + p2;
        }
        return p3;
    }
};
```
