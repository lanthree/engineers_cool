# 74. 搜索二维矩阵

[原题链接](https://leetcode-cn.com/problems/search-a-2d-matrix/)

就是标准的二分搜索:

```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();
        int l = 0, r = m*n-1;
        while (l <= r) {
            int mid = l + (r-l)/2;
            int i = mid/n, j = mid%n;
            if (matrix[i][j] == target) {
                return true;
            } else if (matrix[i][j] > target) {
                r = mid-1;
            } else {
                l = mid+1;
            }
        }
        return false;
    }
};
```
